====================
Welcome!
====================
Calcfix is designed so you don't have to
panic after something screwed up settings,
or graphing. 

By the way, it's pretty easy to use.

====================
Installing
====================
If you haven't already, grab yourself a
copy of TI-Connect CE, connect your
calculator via. the USB cable that came
with your calculator, and drag and drop
the Calcfix programs onto your desktop.

====================
Usage
====================
Using either prgm or Doors CSE, navigate to
a program with the prefix CFX, and open it.
Read the programs section below to get a
sense of what program is what program.

====================
Programs
====================
CFXALL.8xp will fix all the issues it can.
CFXGRPH.8xp will fix all graphing issues (format, window)
CFXWIND.8xp will only fix window issues
CFXGRCL.8xp will only do a garbage collect.
CFXMODE.8xp will only fix mode issues.
